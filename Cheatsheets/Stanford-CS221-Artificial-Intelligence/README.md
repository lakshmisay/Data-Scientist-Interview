# Artificial Intelligence cheatsheets for Stanford's CS 221

## Goal
This repository aims at summing up in the same place all the important notions that are covered in Stanford's CS 221 Artificial Intelligence course, and include **cheatsheets** for each **artificial intelligence field**.
